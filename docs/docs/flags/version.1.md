## NAME

tracee **version** - Print the version number of Tracee

## SYNOPSIS

tracee **version**

## DESCRIPTION

The **version** command displays the current version number of Tracee.

## EXAMPLES

- Print the version:

```console
tracee version
```

- Using the alias:

```console
tracee v
```
